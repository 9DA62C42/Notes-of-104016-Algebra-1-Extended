## English

### Overview

This repository contains the lecture notes for the **Algebra 1/Extended** course (course ID: 104016), authored by **Xiangjun Liu**.

The notes are typeset in LaTeX and compiled into a PDF document. They cover a wide range of topics in linear algebra and abstract algebra, organized as follows:

### Topics Covered

- Some abbreviations and notations
- Fields, finite fields and modular arithmetic
- Rational, real and complex numbers
- Polynomials
- Vectors
- Matrices
- Vector spaces and subspaces
- Span and generating systems
- Linear dependence and independence
- Basis
- Dimension
- The sum of subsets, subspaces and direct sum
- Null space and kernel
- Linear transformations
- Coordinate and matrix representation of linear transformations
- Determinant
- Diagonalization and eigenvalues
- Annihilating polynomials
- Proofs

\section{Some abbreviations and notations}
        \subsection{Abbreviations}        
            \hskip \parindent
            \par Throughout this notes, we will use the following abbreviations:
            \begin{itemize}
                \item s.t. : such that
                \item coeff. : coefficient
                \item iff : if and only if
                \item rep. : representation
                \item l.s. : linear system
                \item l.i. : linear independent
                \item l.d. : linear dependent
                \item l.c. : linear combination
                \item l.o. : linear operator
                \item l.t. : linear transformation
                \item v.s. : vector space
                \item f.d.v.s. : finite dimensional vector space
                \item n-d.v.s. : n-dimensional vector space
                \item alt. : alternating
                \item mult. : multilinear
            \end{itemize}
        \subsection{Notations}
            \hskip \parindent
            \par Throughout this notes, we will use the following notations:
            \begin{itemize}
                \item $\mathbb{N}$ : the set of natural numbers $\{1,2,3,\ldots\}$
                \item $\mathbb{N}_0$ : the set of natural numbers including zero $\{0,1,2,3,\ldots\}$
                \item $\mathbb{Z}$ : the set of integers $\{\ldots,-2,-1,0,1,2,\ldots\}$
                \item $\mathbb{Q}$ : the set of rational numbers
                \item $\mathbb{R}$ : the set of real numbers
                \item $\mathbb{C}$ : the set of complex numbers
                \item $\mathbb{F}$ : a general field
                \item $\mathbb{F}_q$ : a finite field with $q$ elements
                \item $\mathbb{Z}_p$ or $\mathbb{F}_p$ : the finite field of integers modulo a prime number $p$
            \end{itemize}
